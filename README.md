# Cursed Antique Shop

A Python course project that simulates a small antique shop with inventory and shopping cart functionality.

## About
This project was created as part of my Python coursework. The goal was to practice core programming concepts by building an interactive menu-driven program.

## Features
- View available antiques
- Buy an item by name or index
- Remove items from the shopping cart
- Sort inventory alphabetically or in reverse alphabetical order
- Handle invalid input with user-friendly messages

## Skills Demonstrated
- Python
- Lists
- Loops
- Conditionals
- User input handling
- Sorting
- Basic error handling with `try` / `except`
